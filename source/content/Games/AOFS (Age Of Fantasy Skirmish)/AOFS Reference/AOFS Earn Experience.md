---
title: Earn Experience
---
Units earn XP in battle for each of the following things:
- **+1 XP:** Surviving the battle. (Not a knocked out/routed)
- **+1 XP:** Knocked out an enemy unit.
- **+2XP:** Knocked out an enemy hero unit.

Track earned XP on a warband sheet or in the army forge to see when units level up.

**Regular Units:** When a unit earns 5 XP it becomes 5pts more expensive and you must roll one die to see what trait it gets:

| Result | Trait                                                                  |
| ------ | ---------------------------------------------------------------------- |
| 1      | **Agile** - Moves +1" on advance and +2" on rush and charge.           |
| 2      | **Headstrong** - Gets +1 to rolls when taking morale tests.            |
| 3      | **Specialist** - Gets +1 to hit rolls in melee or shooting (pick one). |
| 4      | **Resilient** - Gets +1 to rolls when blocking hits.                   |
| 5      | **Elite** - May re-roll one die of any kind once per round.            |
| 6      | **Fast Learner** - May pick one of any other trait in the list.        |

> [!info] Note
> Each trait (except for Specialist) can only be gained once, so if you roll a trait you already have, keep re-rolling until you get a new trait you don't have yet. 
> 
> Units can't earn more than 30XP, so they can't level up more than 6 times.


**Hero Units:** Follow the same rules as above, but heroes become 15pts more expensive (instead of only 5pts) when they level up and get skill traits on top of regular traits. The first time a hero levels up roll one die and pick one skill set:

| Result | Skill Set            |
| ------ | -------------------- |
| 1-2    | Captain or Support   |
| 3-4    | Fighter or Shooter   |
| 5-6    | Pathfinder or Healer |

Every time the hero levels up, including the first time, roll one die to see what skill trait it gets from its skill set. 

**Captain:** The hero shines as a leader and their men follow with pride.

| Result | Captain Skill Set                                                                                |
| ------ | ------------------------------------------------------------------------------------------------ |
| 1-2    | **Leader** - Friendly units within 6" of the Hero get +1 to moral test rolls.                    |
| 3-4    | **Instigator** - Friendly units that activate within 6” of the Hero get +1 to hit when in melee. |
| 5-6    | **Tactician** - Friendly units that activate within 6” of the Hero get +1 to hit when shooting.  |


**Support:** The hero user powerful tools to buff allies and debuff enemies.

| Result | Support Skill Set                                                                               |
| ------ | ----------------------------------------------------------------------------------------------- |
| 1-2    | **Vanguard** - Friendly units that activate within 6” of the Hero get +4” charge range.         |
| 3-4    | **Scavenger** - Friendly units that activate within 6” of the Hero get +6” range when shooting. |
| 5-6    | **Mastermind** - Enemy units within 6” of the Hero get -1 Defense.                              |

**Fighter:** The hero has perfected close combat techniques beyond compare.

| Result | Fighter Skill Set                                                                               |
| ------ | ----------------------------------------------------------------------------------------------- |
| 1-2    | **Duellist** - Enemy units get -1 to hit in melee against the hero.                             |
| 3-4    | **Fanatic** - The hero gets +1 attack when in melee.                                            |
| 5-6    | **Berserker** - Enemies get +1 to wound effect rolls when taking wounds from the hero in melee. |

**Shooter:** The hero expertly handles all sorts of weapons and equipment.

| Result | Shooter Skill Set                                                                                            |
| ------ | ------------------------------------------------------------------------------------------------------------ |
| 1-2    | **Hunter** - The hero may shoot even after rush actions.                                                     |
| 3-4    | **Destroyer** - The hero gets AP(+1) when shooting.                                                          |
| 5-6    | **Supressor** - Enemies that are shot at by the hero get -1 to hit when shooting until the end of the round. |

**Pathfinder:** The hero is a true master at sneaking and collecting intel.

| Result | Pathfinder Skill Set                                                                      |
| ------ | ----------------------------------------------------------------------------------------- |
| 1-2    | **Runner** - The hero always moves +2” (even in Difficult terrain).                       |
| 3-4    | **Prowler** - Enemy units further than 12” get -1 to hit rolls when shooting at the hero. |
| 5-6    | **Climber** - The hero needs to roll only 2+ when jumping.                                |

**Healer:** The hero mends and protects allies from terrible injuries with ease.

| Result | Healer Skill Set                                                                                    |
| ------ | --------------------------------------------------------------------------------------------------- |
| 1-2    | **Chemist** - Friendly units within 6” of the Hero get -1 to wound effect rolls.                    |
| 3-4    | **Herbalist** - Friendly units within 6” may ignore wounds on a roll of 6+.                         |
| 5-6    | **Warden** - When the hero is activated all friendly units within 6” of the hero stop being shaken. |


> [!info] Note
> Each trait can only be gained once, so if you roll a trait you already have keep re-rolling until you get a new trait you don’t have yet. If a hero already has all 3 traits of its skill set, then it may roll on the table again to get a new second skill set.
